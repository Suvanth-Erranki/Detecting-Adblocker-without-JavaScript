Detecting an adblocker without JavaScript demo:



Adblocker detected!


If you don't see a message saying that an adblocker is present, it means that you aren't using an adblocker!

You can disable JavaScript in your browser's settings and this detection method will still work.





This website detects uBlock Origin, Adblock Plus, Adblock, and many other adblockers. So why does this work?

The truth: There is no actual detection being done by the website. It just claims it detects adblockers. What really happens is the "Adblocker detected!" text is always there even if you aren't using an adblocker. The text is covered by a hidden division element (div) with the id "adsbox". If the user is using an adblocker, the adblocker believes that the div must be an advertisement since the div's id says "adsbox". So the adblocker removes it, thus revealing the message that an adblocker was detected. This can be used to greater measure and this is just a simple demo. Clever right?

The issue is, a lot websites these days usually display their advertisements in a very unethical manner which is where adblock are useful. However, without a way to make income, websites have no choice but are forced to block users using adblockers (which adblockers sometimes unblock). Advertisements should be shown (albeit in a fair manner) as the website has to pay for hosting the site.

How can this be used in normal websites as this is just a demo:

It's not as easy as you would with JavaScript, but you could create a banner at the top of your website and either hide or show a message asking kindly to not use an adblocker. It's really just preference.

Can users remove this message?

Unfortunately, yes. Using inspect element, a bookmarklet, a userscript, or an extension, the user can hide the message. However, if the user has JavaScript enabled, you can just detect if the message is deleted and then use JavaScript to reload your website.

So why did I not use JavaScript?

The user can easily disable JavaScript in site settings and the whole point is for websites to use this method to kindly ask the user to disable their adblocker if present.

P.S. If you are a person who uses an Adblocker consider enabling the acceptable ads option in your adblocker's settings if there is an option and you haven't already done it. If you are an Adblocker developer, please consider adding an acceptable ads option if you haven't already.
